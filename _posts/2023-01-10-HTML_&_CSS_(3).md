# css 파일 따로 쓰기



<br/>


1. 새로 .css 파일을 만들어준다



2. 기존 html 안에 style 태그 내용들을 새로만든 파일에 넣어준다



3. 기존 html 파일에 css 주소를 입력한다



```html

<link href="css파일주소.css" rel="stylesheet>

```



<br/>


# 간단히 태그안에 css 넣는법



<br/>


기존 태그에 style 값들을 넣어준다



스타일속성을 빠르게 적용하고 싶을 때 주로 사용하기 좋다



```html

<h1 style="color: red; font-size: 72px;">제목들</h1>

```
